1. Выпадающие списки (Dropdown)
Выпадающий список (Dropdown) — это элемент интерфейса, который позволяет пользователю выбрать одно значение из предустановленного набора вариантов. Это удобно, когда количество опций велико, и вам не нужно показывать все варианты сразу, а только тот, который выбран.

Как использовать Dropdown в Flet:
ft.Dropdown() — создает выпадающий список.

Аргумент options принимает список объектов ft.dropdown.Option, каждый из которых представляет элемент выпадающего списка.

При выборе элемента можно использовать событие on_change для обработки выбора.
