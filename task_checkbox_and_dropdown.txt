Настройки приложения с выпадающим списком и флажками
Описание: Создайте форму настроек для приложения, в которой пользователь может выбрать язык (с помощью выпадающего списка) и активировать или деактивировать некоторые функции с помощью флажков.

Требования:

Используйте выпадающий список для выбора языка: Русский, Английский, Французский.

Используйте флажки для выбора дополнительных опций, например: "Уведомления", "Тема светлая", "Автосохранение".

Когда пользователь нажимает кнопку "Применить", отображается результат выбора.

----

Переключатель для включения/выключения режима
Описание: Разработайте приложение с переключателем для включения или выключения определенного режима. Например, переключатель для активации режима "Темной темы".

Требования:

Добавьте переключатель с меткой "Темная тема".

После изменения состояния переключателя покажите сообщение, что режим включен или выключен.

Форма для выбора интересов с флажками
Описание: Создайте форму, где пользователь может выбрать свои интересы с помощью флажков. После выбора интересов отобразите их в текстовом поле.

Требования:

Используйте флажки для интересов: Музыка, Спорт, Путешествия, Кулинария.

После выбора интересов отобразите выбранные опции в текстовом поле.

---

Выбор темы с переключателем и выпадающим списком
Описание: Создайте приложение, где пользователь может выбрать тему интерфейса через выпадающий список и включить или выключить дополнительную настройку через переключатель.

Требования:

Используйте выпадающий список для выбора темы: Светлая, Темная, Яркая.

Добавьте переключатель для включения или выключения уведомлений.

После изменения выбора показывайте текст с выбранной темой и состоянием уведомлений.

---

Выбор возраста с выпадающим списком и флажками для активности
Описание: Создайте форму, в которой пользователь выбирает свой возраст через выпадающий список и ставит флажок, подтверждающий свою активность (например, участие в мероприятиях).

Требования:

Используйте выпадающий список для выбора возраста (например, от 18 до 60 лет).

Добавьте флажок "Я активен" для подтверждения участия.

После выбора возраста и активации флажка показывайте соответствующее сообщение.
